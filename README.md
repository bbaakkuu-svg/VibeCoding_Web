# VibeCoding_Web
Web TUTOR-IA
PROMPT 1:
Actua como un Desarrollador Web Senior. Crea una plataforma de Aula Virtual de Programación moderna y responsive. 
Requisitos Técnicos:
- Estructura Modular: Genera tres archivos independientes (index.html, style.css, script.js).
- Navegación Dinámica: Incluye una sección de cuadricula (Grid) para cada lenguaje: Java, SQL, Shell, Git/GitHub,  HTML5 y CSS3. Cada sección debe tener un espacio para incrustar videos de YouTube (iframes) con títulos y descripciones. .
- Buscador Inteligente: Implementa una barra de busqueda funcional con JavaScript que filtre las tarjetas de lenguaje en tiempo real según el nombre introducido.
- Interacción de Usuario: Crea un 'Buzón de sugerencias' con validación de formulario (nombre, email, sugerencias) y una respuesta visual al enviar.
- Estética Visual: Estilo 'Dark Mode' profesional con acentos en colores neón (un color distintivo para cada lenguaje, ej: Naranja para HTML, azul para CSS, rojo para Java). Usa fuentes sans-serif modernas y espaciado generoso (padding/margin).
- Restricción: Código Vanilla (HTML5, CSS3, JS puro). Sin frameworks externos.
